## A small weather application

### Features
* Next hours and next days forecast for selected city
* Auto-complete city search within 20.000 cities

### Built With
* Vuejs 3
* Tailwind Css
* openweathermap API

### Setup
```bash
 npm install
```

### Development
```bash
npx vite
```

### Build for production
```bash
npx vite build
```

### locally preview production
```bash
npx vite preview
```
